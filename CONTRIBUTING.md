# Contributing to PlutoSpace

PlutoSpace is an independent fork of [Pluto.jl](https://github.com/fonsp/Pluto.jl), developed by
Dale Black / [GroupTherapyOrg](https://github.com/GroupTherapyOrg). It is **not** the Pluto project.

- **Issues & ideas about PlutoSpace** (the workspace shell, terminal, remotes, lazy mode, persistence):
  please open them here on [GroupTherapyOrg/PlutoSpace.jl](https://github.com/GroupTherapyOrg/PlutoSpace.jl/issues).
- **Issues about the notebook engine itself** (reactivity, `@bind`, packages, the file format) usually
  belong upstream with [Pluto.jl](https://github.com/fonsp/Pluto.jl) — those are Pluto's, not ours.

Architecture notes for working on this repo live in [.claude/CLAUDE.md](.claude/CLAUDE.md).
